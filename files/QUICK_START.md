# ⚡ QUICK START - Deploy in 5 Minutes

## Choose Your Path (Pick One)

### 🏃 **Path 1: Netlify (Easiest - 2 Minutes)**

1. Open https://app.netlify.com/drop
2. Drag & drop `tekkishodan-website.html` onto the page
3. Your website is LIVE! 🎉
4. Claim your free `.netlify.app` domain
5. (Optional) Add custom domain in Settings

**Pros:** Free, instant, no technical knowledge needed
**Best for:** Quick launch, beginners

---

### 🐙 **Path 2: GitHub Pages (Most Professional - 3 Minutes)**

1. Create GitHub account (https://github.com/signup)
2. Click **"New Repository"** button
3. Name it: `tekkishodan`
4. Upload `tekkishodan-website.html` (rename to `index.html`)
5. Go to **Settings** → **Pages** → Select `main` branch
6. Your site is at: `yourusername.github.io/tekkishodan`

**Pros:** Professional, free SSL, version control
**Best for:** Long-term management, developers

---

### 🐳 **Path 3: Docker (For Servers - 1 Minute)**

```bash
# Copy all files to your server
docker-compose up -d

# Your site runs on http://your-server-ip
```

**Pros:** Enterprise-ready, scalable, containerized
**Best for:** Production servers, teams

---

### 💻 **Path 4: Test Locally (0 Minutes)**

Simply open `tekkishodan-website.html` in your browser!
